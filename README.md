## # 커밋 컨벤션 

| 유형     | 상세                                         |
| -------- | -------------------------------------------- |
| Feat     | 새로운 기능 추가                             |
| Fix      | 버그 수정                                  |
| Refactor | 코드 리팩토링 (기능 변경 없이 구조 개선)          |
| Test     | 테스트 코드 작성                             |
| Chore    | 기타 자잘한 작업 (빌드 설정, 패키지 관리 등)      |
| Docs     | 문서 추가 또는 수정                          |
| Delete   | 불필요한 코드나 파일 삭제                      |
| Build    | 빌드 관련 파일 및 설정 변경                    |
| Style    | css 및 style 수정                           |
